Installation

There are a few icon managers out there. I use LiteIcon (free).

Just drag and drop the .icns files onto their corresponding applications, and then log out and back in to refresh the dock.

I recommend using a 2D dock with this icon set. Instructions are in the README.md.

About

Made with love (and obsession) by Tina as an alternative icon set that is more consistent.

For icon requests, get in touch.

Contribute

The project is on GitHub. If you want to modify any icons or create your own, the vectors of all existing icons as well as a template are available there and in the download link.
